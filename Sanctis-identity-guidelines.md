# SANCTIS Identity Guidelines

To maintain clarity and consistency, all public or academic use of SANCTIS
should adhere to the following:

## Required Attribution
- “Powered by SANCTIS Cognitive Architecture”
- Include link to the GitHub repository
- Include the author name: Luke “Umbraflamma” Yuro

## Layer Identity Preservation
When referencing SANCTIS layers:
- Use the canonical layer names
- Do not rename or replace metaphors without attribution
- Do not reassign layer roles

## Visual / Mascot Consistency
If you illustrate or visualize SANCTIS:
- Keep layer colors/symbols recognizable  
- Do not imply commercial affiliation  
- Derivative mascots must acknowledge original SANCTIS mascots

## Derivative Works
If you build a new architecture inspired by SANCTIS, you must:
- Credit SANCTIS clearly  
- Release under the same license  
- Maintain separation of branding
