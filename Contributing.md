# Contributing to SANCTIS

Thank you for your interest in contributing!

Before submitting changes, please note:

## License Agreement
By contributing, you agree that:
- Your contributions fall under the same CC BY-NC-SA license as SANCTIS.
- You do NOT gain commercial rights.
- You grant a non-exclusive license to the author to use your contributions
  in future SANCTIS versions, including proprietary editions.

## Rules for Contributions
- No commercial components.  
- No integration code for proprietary LLMs without permission.  
- No redefinition of SANCTIS layer identities.  
- Changes must remain compatible with SANCTIS architectural philosophy.

## How to Contribute
1. Fork the repo  
2. Make changes  
3. Submit a PR with a clear description  
4. Wait for review  

Thank you for helping evolve SANCTIS responsibly.
